# AlarmClock

***Clone of native iOS Alarm clock app***

**General**
* No Storyboard (SnapKit)
* MVC

**Data store**
* UserDefaults
* CoreData

**Animation**
* CALayer, CAShapeLayer
* ImageContext
* UIBasicAnimation

**Collection feature**
* No TableView (CollectionView with Compositional Layout)
* UIDiffableDataSource
* Modern Cell Configuration (Content Configuration)

**Other**
* Subscripts
* Hashable
* Associated values
