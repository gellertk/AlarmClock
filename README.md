# AlarmClock

***Clone of native iOS Alarm clock app***

**General**
* No Storyboard (SnapKit)
* MVC

**Data stores**
* UserDefaults
* CoreData

**Animation**
* CALayer, CAShapeLayer
* ImageContext
* UIBasicAnimation

**Collection features**
* UIDiffableDataSource
* Modern Cell Configuration (contentConfiguration)

**System**
* RunLoop
